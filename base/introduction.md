# introduction
ES 是什么？
1、全文搜索引擎
2、一个分布式的实时文档存储，每个字段 可以被索引与搜索
3、一个分布式实时分析搜索引擎
4、能胜任上百个服务节点的扩展，并支持 PB 级别的结构化或者非结构化数据
